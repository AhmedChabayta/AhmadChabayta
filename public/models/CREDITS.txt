PUT YOUR TREE MODEL HERE  ->  public/models/tree.glb

The renderer auto-centers, scales, lights, shadows, winds and
scroll-frames whatever .glb is at that exact path. Until the file
exists, the scene stays empty and the site is unaffected.

Requirements:
  - File name must be exactly:  tree.glb
  - Format: glTF Binary (.glb), UNCOMPRESSED or meshopt-compressed.
    Draco-compressed .glb is NOT supported yet (tell me and I'll
    add a Draco decoder).
  - Any real-world scale / orientation is fine (auto-normalized).
  - Keep it reasonably sized (ideally < ~25 MB for fast loads).

Replace this model anytime by committing a new tree.glb here.
